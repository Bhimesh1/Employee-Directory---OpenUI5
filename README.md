# Employee Directory - OpenUI5

A simple employee directory application built with OpenUI5. The project demonstrates XML views, MVC structure, data binding, filtering, and routing using a lightweight browser-based setup.

## Tech Stack

* OpenUI5
* JavaScript
* XML Views
* JSONModel
* Node.js static server

## Run Locally

Start a local static server:

```bash
npx http-server -p 8085
```

Open the app:

```txt
http://localhost:8085
```

## Project Structure

```txt
employee-directory-openui5/
  index.html
  Component.js
  manifest.json
  view/
    App.view.xml
  controller/
    App.controller.js
```

## Current Status

The application shell is set up with OpenUI5, a component-based bootstrap, manifest configuration, and an XML root view.
